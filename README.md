# Mahatm - Company Website

Welcome to the Mahatm company website repository. This website is hosted on GitHub Pages.

## Overview

This is the official website for Mahatm, showcasing our company's services and providing information about who we are.

## Features

- Responsive design that works on desktop and mobile devices
- Clean, modern user interface
- Sections for Home, About, Services, and Contact

## Deployment

This website is designed to be hosted on GitHub Pages. To enable GitHub Pages:

1. Go to the repository Settings
2. Navigate to the "Pages" section
3. Under "Source", select the branch you want to deploy (usually `main`)
4. Click Save

The website will be available at `https://<username>.github.io/mahatm/`

## Local Development

To view the website locally, simply open the `index.html` file in your web browser.
